## 	Getting Started

<!-- Run Locally -->
### Run Locally

Clone the project

```bash
  git clone https://github.com/natainditama/auctionx.git
```

Go to the project directory

```bash
  cd auctionx
```

Install dependencies

```bash
  npm install
```

Build resources & assets

```bash
  npx gulp build
```
